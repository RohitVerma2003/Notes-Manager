## ğŸ”§ Setup Instructions

1. Clone the repository  
2. Navigate to both frontend and backend folders  
3. Install dependencies using npm install  
4. In the backend, create a `.env` file and add your MongoDB URI:  
   `MONGO_DB_URI=your_connection_string`  
5. Run the backend server using `npm run dev`  
6. Start the Expo app from the frontend folder using `npx expo start`
